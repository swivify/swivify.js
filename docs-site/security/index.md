# Security Best Practices

- Use environment variables for all secrets and credentials
- Keep dependencies updated (use `npm audit`/`pnpm audit`)
- Validate all user input (use libraries like zod or joi)
- Use HTTPS in production and secure cookies
- Restrict CORS as needed for your API
- Regularly review and update your config and plugins
- Use Inspector to monitor logs and API health
